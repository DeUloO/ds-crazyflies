class crazyflies.crazyflie.Crazyflie(node, id, channel, initial_position, type)

Bases: ConsoleClient, EmergencyClient, GenericCommanderClient, HighLevelCommanderClient, LoggingClient, RPYTCommanderClient

Represents a crazyflie.

Allows user to interchangeably use different crazyflie implementations (currently Webots or Real Hardware)

Initializes a Console client.

Pass a callback in order to receive message Strings sent from the crazyflie

Parameters:

  • node (Node) – A rclpy Node to attach the subscriber to

  • prefix (str) – The ros prefix for the crazyflie (e.g. cfXX)

  • callback (Callable[[str],]) – A callback which gets called when a message gets received

  • id (int)

  • channel (int)

  • initial_position (List[float])

  • type (CrazyflieType)
