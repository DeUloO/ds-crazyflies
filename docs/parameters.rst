.. _parameters:

Parameters
============

.. toctree:: 
    :maxdepth: 100

The crazyflie `parameters <https://www.bitcraze.io/documentation/repository/crazyflie-firmware/master/api/params/>`_ are directly mapped and connected to the equivalent ROS Parameters.
With ``ros2 parameter set ..`` the specific parameter will also be updated on the crazyflie.
